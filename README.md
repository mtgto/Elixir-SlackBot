Elixir-SlackBot
========

A slack bot application by Elixir. You can extend it by plugins.

## Description

`Elixir-SlackBot` is an Elixir application to communicate with Slack RTM API.

## Features

- Auto reconnection
- Plugin structure

## Requirement

- Elixir

## Usage

1. Generate a Slack API token.
1. Create new plugin.
1. Configure the app.
1. Run.
